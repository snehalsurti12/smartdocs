Works With Your Workflow

Generate documents however you need them.

Record Page Button

Add the SmartDocs component to any record page. Users click, get a PDF attached to the record.

Salesforce Flows

Use the "Generate SmartDocs Document" action in any Flow — screen flows, record-triggered, scheduled.

Agentforce Actions

AI agents generate documents autonomously. "Send me my invoice" → PDF created and delivered.

Batch Processing

Generate hundreds of documents in batch jobs. Monthly statements, renewal letters, mass mailings.

REST API

POST template ID + data, get PDF back. Integrate from any system — not just Salesforce.
