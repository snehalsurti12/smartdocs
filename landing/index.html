ENTERPRISE READY

Built for regulated industries

Multi-tenant isolation, zero data retention, PII protection. Ready for compliance review on day one.

Tenant Isolation

Every API key scopes to one tenant. Templates, renders, and storage are completely siloed. No cross-tenant access — ever.

Zero Retention

Client data lives in memory only during render. No disk writes, no logs, no cache. PII fields are tagged and masked in audit trails.

Encrypted at Rest

Storage credentials encrypted with AES-256-GCM. TLS enforced. Each render runs in an isolated browser context, destroyed after use.

Audit Trail

Every render logged with tenant, template, version, and timestamp. PII values excluded from logs. Full compliance reporting.

BYOS

Bring Your Own Storage. Connect your S3, GCS, or Azure bucket. Generated PDFs go directly to your infrastructure — never ours.

Users & Roles

Four roles: Author, Reviewer, Publisher, Admin. Invite-only access. Project-scoped membership. Scoped API keys for programmatic access.
