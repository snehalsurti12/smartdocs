Built for real documents

Not another toy PDF generator. SmartDocs handles enterprise communications — statements, invoices, reports, event passes.

Visual Editor

Drag components onto a canvas. Resize, align, snap to guides. See live data preview. Undo/redo everything.

Charts

Bar, line, pie, doughnut — all SVG, all data-bound. Render perfectly in PDF print output.

Barcodes & QR

Real Code 128 encoding. QR codes up to version 10 with Reed-Solomon error correction. Scannable output.

Salesforce Native

Point-and-click field mapping. LWC button, Flow action, Agentforce ready. No Connected App or OAuth.

Zero Retention

Data exists only in memory during render. No logs, no cache, no disk writes. PII field tagging built in.

Fast

Chromium browser pool. ~2 seconds per PDF. Batch processing for hundreds of documents.

One API call.
One PDF back.

POST your template ID and data. Get a print-ready PDF in ~2 seconds. Works from Salesforce, Node.js, Python, curl — anything that speaks HTTP.

# Generate a PDF from a template curl -X POST /api/render \ -H "Authorization: Bearer sk_live_..." \ -d '{ "templateId": "tmpl_invoice_001", "data": { "company": { "name": "Acme Corp" }, "items": [ { "name": "Widget", "qty": 10 } ] } }' # → 200 OK (application/pdf, 28KB)
