# Security Policy

## Supported Versions

This project is currently in `0.x (alpha)` and does not guarantee full backward compatibility.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report privately to maintainers with:
- clear description of impact
- affected files/paths
- reproduction steps
- suggested mitigation (if known)

We will acknowledge receipt and provide status updates as triage progresses.
